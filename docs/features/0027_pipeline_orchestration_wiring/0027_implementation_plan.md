# 0027 Pipeline Orchestration Wiring

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire the three-stage pipeline (scan → discover → prove) so that stage completion triggers the next stage, discover proactively consumes scan results by default, and prove gracefully degrades when no findings exist.

**Architecture:** The pipeline data model and state machine exist (0026) but are disconnected. Five gaps: (1) `AdvanceStage` is never called after audit completion, (2) `AdvanceStage` updates status but doesn't create the next audit, (3) `pipelineSvc` is instantiated with `nil` discoverSvc, (4) discover ignores scan results, (5) prove exits instead of continuing without findings. This feature wires everything together and adds a new `ignore_scan_results` config option on discover.

**Tech Stack:** Go 1.24+ (backend), Python 3.12+ (agents), FastAPI, httpx, PostgreSQL/SQLite, Isabelle/HOL (formal verification)

**Key Behavioral Contract:**
- **Scan agents** run independently — no changes.
- **Discover agent** runs independently. By default, if `source_path` is available, it proactively fetches scan results from the backend and uses them to seed route discovery. New config option `ignore_scan_results` (default: `false`) skips this. When called via pipeline, scan findings are injected into config directly.
- **Prove agent** runs the full verify pipeline when findings exist. When no findings exist, it proceeds with discovery + proactive API probing instead of exiting.
- **Pipeline orchestrator** auto-cascades: scan completion triggers discover, discover completion triggers prove. Each stage audit is created and launched automatically.

**Methodology — Proof-First for State Machine, Test-First for I/O:**
- The pipeline state machine (transitions, stage expansion, terminal states) is **proven correct in Isabelle BEFORE implementation**. Isabelle extracts a verified oracle binary. Go implementation is written to pass conformance tests against this oracle.
- The I/O-heavy code (HTTP handlers, SSE streaming, agent dispatch, database writes) is **test-first (TDD)** — proofs are impractical for I/O.
- The verification directory lives inside the project but is fully independent — no imports from Go/Python, runnable standalone via `make verify`.

```isabelle
(* verification/isabelle/Pipeline_State.thy *)
theory Pipeline_State
  imports Main
begin

section \<open>Pipeline Status Type\<close>

datatype pipeline_status =
    Pending
  | ScanRunning
  | DiscoverRunning
  | ProveRunning
  | Completed
  | Failed

datatype stage = Scan | Discover | Prove

datatype audit_outcome = AuditCompleted | AuditFailed

record pipeline =
  p_status :: pipeline_status
  p_stages :: "stage list"
  p_index  :: nat

section \<open>Transition Functions\<close>

fun stage_to_running :: "stage \<Rightarrow> pipeline_status" where
  "stage_to_running Scan = ScanRunning"
| "stage_to_running Discover = DiscoverRunning"
| "stage_to_running Prove = ProveRunning"

(* Safe list access — returns None for out-of-bounds *)
fun nth_safe :: "'a list \<Rightarrow> nat \<Rightarrow> 'a option" where
  "nth_safe [] _ = None"
| "nth_safe (x # xs) 0 = Some x"
| "nth_safe (x # xs) (Suc n) = nth_safe xs n"

(* The core state transition — this IS the specification.
   Go implementation must match this exactly. *)
fun advance :: "pipeline \<Rightarrow> audit_outcome \<Rightarrow> pipeline" where
  "advance p outcome =
    (if p_status p = Completed then p
     else if p_status p = Failed then p
     else if p_status p = Pending then p  \<comment> \<open>guard: only running states can advance\<close>
     else case outcome of
       AuditFailed \<Rightarrow> p\<lparr> p_status := Failed \<rparr>
     | AuditCompleted \<Rightarrow>
         (case nth_safe (p_stages p) (Suc (p_index p)) of
            None \<Rightarrow> p\<lparr> p_status := Completed \<rparr>
          | Some next \<Rightarrow> p\<lparr> p_status := stage_to_running next,
                              p_index := Suc (p_index p) \<rparr>))"

section \<open>Proofs — Terminal States\<close>

theorem completed_is_absorbing:
  "p_status p = Completed \<Longrightarrow> advance p outcome = p"
  by simp

theorem failed_is_absorbing:
  "p_status p = Failed \<Longrightarrow> advance p outcome = p"
  by simp

theorem pending_is_guarded:
  "p_status p = Pending \<Longrightarrow> advance p outcome = p"
  by simp

section \<open>Proofs — Failure Propagation\<close>

theorem failure_always_reaches_failed:
  "\<lbrakk> p_status p \<noteq> Completed; p_status p \<noteq> Failed; p_status p \<noteq> Pending \<rbrakk>
   \<Longrightarrow> p_status (advance p AuditFailed) = Failed"
  by (cases "p_status p") auto

theorem failure_preserves_stages:
  "p_stages (advance p AuditFailed) = p_stages p"
  by (cases "p_status p") auto

theorem failure_preserves_index:
  "p_index (advance p AuditFailed) = p_index p"
  by (cases "p_status p") auto

section \<open>Proofs — Forward Progress\<close>

theorem advance_never_decreases_index:
  "p_index (advance p AuditCompleted) \<ge> p_index p"
  by (cases "p_status p"; simp split: option.splits)

theorem advance_increments_by_at_most_one:
  "p_index (advance p AuditCompleted) \<le> Suc (p_index p)"
  by (cases "p_status p"; simp split: option.splits)

section \<open>Proofs — Full Pipeline Lifecycle\<close>

theorem three_stage_pipeline_completes:
  "let p0 = \<lparr> p_status = ScanRunning, p_stages = [Scan, Discover, Prove], p_index = 0 \<rparr>;
       p1 = advance p0 AuditCompleted;
       p2 = advance p1 AuditCompleted;
       p3 = advance p2 AuditCompleted
   in p_status p3 = Completed"
  by simp

theorem three_stage_passes_through_all_states:
  "let p0 = \<lparr> p_status = ScanRunning, p_stages = [Scan, Discover, Prove], p_index = 0 \<rparr>;
       p1 = advance p0 AuditCompleted;
       p2 = advance p1 AuditCompleted;
       p3 = advance p2 AuditCompleted
   in p_status p0 = ScanRunning
    \<and> p_status p1 = DiscoverRunning
    \<and> p_status p2 = ProveRunning
    \<and> p_status p3 = Completed"
  by simp

theorem three_stage_needs_exactly_three_completions:
  "let p0 = \<lparr> p_status = ScanRunning, p_stages = [Scan, Discover, Prove], p_index = 0 \<rparr>;
       p1 = advance p0 AuditCompleted;
       p2 = advance p1 AuditCompleted
   in p_status p2 = ProveRunning \<and> p_status p2 \<noteq> Completed"
  by simp

section \<open>Proofs — Idempotency\<close>

theorem double_advance_completed_is_identity:
  "p_status p = Completed \<Longrightarrow> advance (advance p o1) o2 = p"
  by simp

theorem double_advance_failed_is_identity:
  "p_status p = Failed \<Longrightarrow> advance (advance p o1) o2 = p"
  by simp

section \<open>Proofs — Status Closure (no invalid states reachable)\<close>

theorem status_remains_valid:
  "p_status (advance p outcome) \<in>
   {Pending, ScanRunning, DiscoverRunning, ProveRunning, Completed, Failed}"
  by (cases "p_status p"; cases outcome; simp split: option.splits)

section \<open>Proofs — No Regression\<close>

(* Once past ScanRunning, can never return to it *)
theorem no_return_to_scan_running:
  "\<lbrakk> p_status p \<in> {DiscoverRunning, ProveRunning, Completed, Failed} \<rbrakk>
   \<Longrightarrow> p_status (advance p outcome) \<noteq> ScanRunning"
  by (cases "p_status p"; cases outcome; simp split: option.splits)

section \<open>Code Extraction\<close>

export_code advance stage_to_running nth_safe
  in Scala module_name PipelineVerified file_prefix pipeline_verified

end
```

```isabelle
(* verification/isabelle/Stage_Expansion.thy *)
theory Stage_Expansion
  imports Main
begin

section \<open>Stage Expansion Specification\<close>

datatype stage = Scan | Discover | Prove

(* Prerequisites: prove requires scan + discover; discover requires scan when source exists *)
fun prereqs :: "stage \<Rightarrow> bool \<Rightarrow> stage list" where
  "prereqs Scan _ = []"
| "prereqs Discover True = [Scan]"        \<comment> \<open>with source: scan first\<close>
| "prereqs Discover False = []"           \<comment> \<open>URL-only: no prerequisites\<close>
| "prereqs Prove _ = [Scan, Discover]"    \<comment> \<open>prove always needs both\<close>

(* Canonical ordering for determinism *)
definition stage_order :: "stage \<Rightarrow> nat" where
  "stage_order s = (case s of Scan \<Rightarrow> 0 | Discover \<Rightarrow> 1 | Prove \<Rightarrow> 2)"

(* Expand requested stages with prerequisites, deduplicated, in canonical order *)
fun expand_stages :: "stage list \<Rightarrow> bool \<Rightarrow> stage list" where
  "expand_stages requested has_source =
    (let all_needed = concat (map (\<lambda>s. prereqs s has_source @ [s]) requested);
         deduped = remdups all_needed
     in sort_key stage_order deduped)"

section \<open>Proofs\<close>

theorem prove_expands_to_three_with_source:
  "expand_stages [Prove] True = [Scan, Discover, Prove]"
  by (simp add: stage_order_def)

theorem prove_expands_to_three_without_source:
  "expand_stages [Prove] False = [Scan, Discover, Prove]"
  by (simp add: stage_order_def)

theorem discover_with_source_includes_scan:
  "Scan \<in> set (expand_stages [Discover] True)"
  by (simp add: stage_order_def)

theorem discover_without_source_excludes_scan:
  "expand_stages [Discover] False = [Discover]"
  by (simp add: stage_order_def)

theorem scan_only_stays_scan:
  "expand_stages [Scan] True = [Scan]"
  by (simp add: stage_order_def)

theorem expansion_preserves_order:
  "\<forall>i j. i < j \<and> j < length (expand_stages req hs)
   \<longrightarrow> stage_order (expand_stages req hs ! i) \<le> stage_order (expand_stages req hs ! j)"
  by (simp add: sorted_sort_key)

theorem expansion_has_no_duplicates:
  "distinct (expand_stages req hs)"
  by (simp add: distinct_remdups)

theorem expansion_is_idempotent:
  "expand_stages (expand_stages req hs) hs = expand_stages req hs"
  by (simp add: stage_order_def)

export_code expand_stages stage_order
  in Scala module_name StageExpansionVerified file_prefix stage_expansion_verified

end
```

```scala
// verification/oracle/src/main/scala/OracleCLI.scala
package vulture

import com.google.gson.{Gson, JsonObject}

/** CLI wrapper around Isabelle-extracted verified functions.
  *
  * Usage:
  *   pipeline-oracle advance '{"status":"scan_running","stages":["scan","discover","prove"],"index":0}' completed
  *   pipeline-oracle expand  '{"stages":["prove"],"has_source":true}'
  *
  * Output: JSON with the result.
  * Exit code 0 = success, 1 = invalid input.
  */
object OracleCLI {
  private val gson = new Gson()

  def main(args: Array[String]): Unit = {
    if (args.length < 2) {
      System.err.println("Usage: pipeline-oracle <command> <json> [outcome]")
      System.exit(1)
    }

    val command = args(0)
    val json = gson.fromJson(args(1), classOf[JsonObject])

    command match {
      case "advance" =>
        val status = parseStatus(json.get("status").getAsString)
        val stages = parseStages(json.getAsJsonArray("stages"))
        val index = json.get("index").getAsInt
        val outcome = parseOutcome(args(2))

        val pipeline = PipelineVerified.Pipeline(status, stages, index)
        val result = PipelineVerified.advance(pipeline, outcome)

        val out = new JsonObject()
        out.addProperty("status", statusToString(result.p_status))
        out.addProperty("index", result.p_index)
        println(gson.toJson(out))

      case "expand" =>
        val stages = parseStages(json.getAsJsonArray("stages"))
        val hasSource = json.get("has_source").getAsBoolean

        val result = StageExpansionVerified.expand_stages(stages, hasSource)

        val out = new JsonObject()
        out.add("stages", gson.toJsonTree(result.map(stageToString).toArray))
        println(gson.toJson(out))

      case _ =>
        System.err.println(s"Unknown command: $command")
        System.exit(1)
    }
  }

  // ... parseStatus, parseStages, parseOutcome, statusToString, stageToString helpers
  // (straightforward enum mapping between JSON strings and Isabelle datatypes)
}
```

```scala
// verification/oracle/src/main/scala/PipelineSimulationOracle.scala
package vulture

import com.google.gson._
import java.io.FileReader
import scala.jdk.CollectionConverters._

/**
 * Combines:
 *   1. Isabelle-proven state machine (PipelineVerified.advance)
 *   2. Ground truth manifest (what the simulated target has)
 *
 * Produces: expected execution trace for the full pipeline.
 * State transitions are PROVEN. Finding expectations are from manifest ground truth.
 */
object PipelineSimulationOracle {

  case class StageExpectation(
    stage: String,
    statusBefore: String,
    statusAfter: String,
    minFindings: Int,
    mustFindVulns: List[String],
    mustFindEndpointsMin: Int,
    mustNotFindVulns: List[String]
  )

  case class ExpectedTrace(stages: List[StageExpectation])

  def generateTrace(manifestPath: String): ExpectedTrace = {
    val root = new Gson().fromJson(new FileReader(manifestPath), classOf[JsonObject])
    val exp = root.getAsJsonObject("pipeline_expectations")
    val vulns = root.getAsJsonObject("vulnerabilities")

    val mustFindIds = vulns.getAsJsonArray("must_find").asScala
      .map(_.getAsJsonObject.get("id").getAsString).toList
    val mustNotFindIds = vulns.getAsJsonArray("must_not_find").asScala
      .map(_.getAsJsonObject.get("id").getAsString).toList

    // Use PROVEN state machine to compute transitions
    val stages = List(PipelineVerified.Scan, PipelineVerified.Discover, PipelineVerified.Prove)
    var pipeline = PipelineVerified.Pipeline(PipelineVerified.ScanRunning, stages, BigInt(0))

    val scanExp = exp.getAsJsonObject("scan_to_discover")
    val discoverExp = exp.getAsJsonObject("discover_to_prove")
    val proveExp = exp.getAsJsonObject("prove_completion")

    val traceStages = List(
      {
        val before = Util.statusStr(pipeline.p_status)
        pipeline = PipelineVerified.advance(pipeline, PipelineVerified.AuditCompleted)
        StageExpectation(
          stage = "scan", statusBefore = before, statusAfter = Util.statusStr(pipeline.p_status),
          minFindings = if (scanExp.get("scan_must_produce_findings").getAsBoolean) 1 else 0,
          mustFindVulns = mustFindIds,
          mustFindEndpointsMin = 0,
          mustNotFindVulns = mustNotFindIds
        )
      },
      {
        val before = Util.statusStr(pipeline.p_status)
        pipeline = PipelineVerified.advance(pipeline, PipelineVerified.AuditCompleted)
        StageExpectation(
          stage = "discover", statusBefore = before, statusAfter = Util.statusStr(pipeline.p_status),
          minFindings = 0,
          mustFindVulns = List(),
          mustFindEndpointsMin = discoverExp.get("discover_must_find_endpoints_min").getAsInt,
          mustNotFindVulns = List()
        )
      },
      {
        val before = Util.statusStr(pipeline.p_status)
        pipeline = PipelineVerified.advance(pipeline, PipelineVerified.AuditCompleted)
        StageExpectation(
          stage = "prove", statusBefore = before, statusAfter = Util.statusStr(pipeline.p_status),
          minFindings = proveExp.get("prove_must_verify_min").getAsInt,
          mustFindVulns = List(),
          mustFindEndpointsMin = 0,
          mustNotFindVulns = List()
        )
      }
    )

    ExpectedTrace(traceStages)
  }

  def toJson(trace: ExpectedTrace): String = {
    new GsonBuilder().setPrettyPrinting().create().toJson(trace)
  }
}

private object Util {
  def statusStr(s: PipelineVerified.Pipeline_status): String = s match {
    case PipelineVerified.Pending => "pending"
    case PipelineVerified.ScanRunning => "scan_running"
    case PipelineVerified.DiscoverRunning => "discover_running"
    case PipelineVerified.ProveRunning => "prove_running"
    case PipelineVerified.Completed => "completed"
    case PipelineVerified.Failed => "failed"
  }
}
```

```go
func (s *pipelineService) advanceToNextStage(pipeline *model.Pipeline, completedAuditID string) error {
	currentStage := s.currentStage(pipeline, completedAuditID)
	nextStage := s.nextStage(pipeline.Stages, currentStage)

	// Idempotency guard: if pipeline already advanced past this stage, no-op.
	// Prevents duplicate audit creation on retries or races.
	expectedStatus := stageToRunning(currentStage)
	if pipeline.Status != expectedStatus {
		return nil
	}

	if nextStage == "" {
		now := time.Now().UTC()
		pipeline.Status = model.PipelineStatusCompleted
		pipeline.CompletedAt = &now
		return s.repo.UpdatePipeline(pipeline)
	}

	cfg, err := s.GetStageAuditConfig(pipeline, nextStage)
	if err != nil {
		return fmt.Errorf("build stage config for %s: %w", nextStage, err)
	}

	types := stageAuditTypes(nextStage, pipeline.Config)
	audit, err := s.auditSvc.Create(&model.AuditRequest{
		SourceID: pipeline.SourceID,
		Types:    types,
		Config:   cfg,
	})
	if err != nil {
		return fmt.Errorf("create %s audit: %w", nextStage, err)
	}

	setStageAuditID(pipeline, nextStage, audit.ID)
	pipeline.Status = stageToRunning(nextStage)

	if err := s.repo.UpdatePipeline(pipeline); err != nil {
		return fmt.Errorf("update pipeline: %w", err)
	}

	// Trigger agent execution for the new audit (see Task 2b)
	if s.runner != nil {
		s.runner.RunPipelineStage(audit.ID)
	}
	return nil
}

func setStageAuditID(pipeline *model.Pipeline, stage, auditID string) {
	switch stage {
	case "scan":
		pipeline.ScanAuditID = auditID
	case "discover":
		pipeline.DiscoverAuditID = auditID
	case "prove":
		pipeline.ProveAuditID = auditID
	}
}

func stageAuditTypes(stage string, pipelineConfig json.RawMessage) []string {
	switch stage {
	case "discover":
		return []string{"discover"}
	case "prove":
		return []string{"prove"}
	case "scan":
		var cfg map[string]interface{}
		if json.Unmarshal(pipelineConfig, &cfg) == nil {
			if raw, ok := cfg["types"]; ok {
				if arr, ok := raw.([]interface{}); ok {
					types := make([]string, 0, len(arr))
					for _, v := range arr {
						if s, ok := v.(string); ok && s != "prove" && s != "discover" {
							types = append(types, s)
						}
					}
					if len(types) > 0 {
						return types
					}
				}
			}
		}
		return config.ScanAgentTypes()
	default:
		return []string{stage}
	}
}
```

```go
func (h *StreamHandler) runPipelineAudit(auditID string) {
	audit, err := h.auditSvc.Get(auditID)
	if err != nil {
		log.Printf("[pipeline] get audit %s: %v", auditID, err)
		return
	}

	var source *model.Source

	if audit.SourceID != "" {

		if err != nil {
			log.Printf("[pipeline] source %s error: %v", audit.SourceID, err)
			return
		}

	}
```

```go
func consumeEventsNoSSE(eventCh <-chan *model.AgUIEvent, auditID string) ([]model.Finding, map[string]int, []model.ProveResult) {
	var findings []model.Finding
	var deltaFindings []model.Finding
	var proveResults []model.ProveResult
	scores := map[string]int{}
	fpLookup := map[string]string{}
	for evt := range eventCh {
		processEvent(evt, auditID, &findings, &deltaFindings, &proveResults, scores, fpLookup)
	}
	if len(findings) == 0 && len(deltaFindings) > 0 {
		findings = deltaFindings
	}
	findings = deduplicateCrossAgent(findings)
	return findings, scores, proveResults
}
```

```go
func (s *pipelineService) GetStageAuditConfig(pipeline *model.Pipeline, stage string) (json.RawMessage, error) {
	var cfg map[string]interface{}
	if err := json.Unmarshal(pipeline.Config, &cfg); err != nil {
		cfg = make(map[string]interface{})
	}

	switch stage {
	case "discover":
		cfg["target_url"] = pipeline.TargetURL
		if pipeline.ScanAuditID != "" && s.auditSvc != nil {
			scanAudit, err := s.auditSvc.Get(pipeline.ScanAuditID)
			if err == nil && scanAudit != nil && len(scanAudit.Findings) > 0 {
				// Cap at 500 findings to keep config payload reasonable for large codebases.
				// Discover only uses file_path for route extraction, so volume matters less than coverage.
				limit := len(scanAudit.Findings)
				if limit > 500 {
					limit = 500
				}
				summaries := make([]map[string]string, 0, limit)
				for _, f := range scanAudit.Findings[:limit] {
					summaries = append(summaries, map[string]string{
						"title":     f.Title,
						"file_path": f.FilePath,
						"category":  f.Category,
						"severity":  string(f.Severity),
					})
				}
				cfg["scan_findings"] = summaries
			}
		}
	case "prove":
		cfg["staging_url"] = pipeline.TargetURL
		if pipeline.DiscoverAuditID != "" && s.discoverSvc != nil {
			dr, err := s.discoverSvc.GetResultByAuditID(pipeline.DiscoverAuditID)
			if err == nil && dr != nil {
				cfg["site_map_json"] = dr.SiteMapJSON
			}
		}
	}

	data, err := json.Marshal(cfg)
	if err != nil {
		return nil, fmt.Errorf("marshal stage config: %w", err)
	}
	return json.RawMessage(data), nil
}
```

```python
import re

_ROUTE_PREFIXES = {"api", "routes", "views", "endpoints", "controllers", "handlers", "pages"}
_EXCLUDED_PREFIXES = {"lib", "utils", "helpers", "test", "tests", "config", "scripts", "migrations"}


def _extract_routes_from_findings(findings: list[dict[str, Any]]) -> list[str]:
    """Extract candidate API routes from scan finding file paths."""
    routes: list[str] = []
    seen: set[str] = set()
    for f in findings:
        fp = f.get("file_path", "")
        if not fp:
            continue
        fp = fp.lstrip(".")
        if not fp.startswith("/"):
            fp = "/" + fp
        parts = fp.split("/")
        for i, part in enumerate(parts):
            lower = re.sub(r"\.\w+$", "", part.lower())
            if lower in _ROUTE_PREFIXES:
                route = "/" + "/".join(parts[i:])
                route = re.sub(r"\.\w+$", "", route)
                if route not in seen:
                    seen.add(route)
                    routes.append(route)
                break
    return routes
```

```python
def _probe_discovered_endpoints(
    loop: asyncio.AbstractEventLoop,
    emitter: AgUiEventEmitter,
    staging_url: str,
) -> Generator[str, None, None]:
    """Probe discovered API endpoints for security issues.

    Used by _run_probe_only (no-findings path). _run_prove_pipeline keeps its
    own inline version because it needs to update counts["verified"] per finding.
    """
    full_site = load_cached_discovery(staging_url)
    if not full_site or not full_site.api_endpoints:
        yield emitter.text_message("No endpoints discovered for probing")
        return

    yield emitter.text_message(
        f"Probing {len(full_site.api_endpoints)} discovered API endpoints..."
    )
    try:
        api_findings = loop.run_until_complete(
            probe_api_endpoints(
                staging_url, full_site.api_endpoints,
                full_site.forms, None,
            )
        )
        if api_findings:
            yield emitter.text_message(f"API probing found {len(api_findings)} issues")
            for af in api_findings:
                yield emitter.proof_result_event(af["id"], af["status"], af["evidence"], 1)
        else:
            yield emitter.text_message("API probing: no issues found")
    except Exception as exc:
        logger.warning("API probing failed: %s", exc)
        yield emitter.text_message(f"API probing skipped: {exc}")
```

```python
def _run_probe_only(
    loop: asyncio.AbstractEventLoop,
    emitter: AgUiEventEmitter,
    staging_url: str,
    source_path: str,
    *,
    schemas: dict,
    capabilities: TargetCapabilities,
) -> Generator[str, None, None]:
    """Discovery + API probing when no scan findings to verify."""
    bg_discovery = _BackgroundDiscovery(
        staging_url, source_path=source_path, schemas=schemas,
    )
    bg_discovery.start()
    yield emitter.text_message("Running discovery for proactive probing...")
    bg_discovery.wait()

    if bg_discovery.result:
        yield emitter.text_message(
            f"Discovery: {len(bg_discovery.result.urls)} URLs, "
            f"{len(bg_discovery.result.api_endpoints)} API endpoints"
        )

    yield from _probe_discovered_endpoints(loop, emitter, staging_url)

    yield emitter.proof_summary_event(
        total=0, verified=0, not_reproduced=0, inconclusive=0, skipped=0,
    )
    yield emitter.run_finished("completed")
```
